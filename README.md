# HR Data Analysis Project

## Overview
This project analyzes an HR dataset to understand employee behavior, performance, and attrition patterns using statistical methods and data visualization techniques.

## Tools Used
- Python (Google Colab)
- Tableau Public

## Dataset
- HR_comma_sep.csv
- Contains 14,995 employee records with attributes such as satisfaction level, evaluation score, number of projects, working hours, department, and salary.

## Analysis Performed
- Data Object Classification (Quantitative, Categorical, Binary)
- Statistical Analysis:
  - Mean, Median, Mode
  - Variance, Standard Deviation, Range, Interquartile Range (IQR)
- Data Visualization:
  - Histogram
  - Scatter Plot
  - Bar Chart

## Tableau Dashboard
Dashboard Link: https://public.tableau.com/app/profile/hritik.kumar2872/viz/HRAnalysis_17748942349560/Dashboard1

## Key Insights
- Employees with low satisfaction levels are more likely to leave the company.
- High working hours combined with high evaluation scores indicate possible workload pressure.
- Certain departments exhibit higher attrition rates.
- Salary levels influence employee satisfaction and retention.
